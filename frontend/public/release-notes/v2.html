Delivery Tracker – Release Notes v2

Release Date Mar 6, 2026

Release Notes Index v3 v1 Summary Key Updates How to Use Notes

Summary

v2 focuses on usability, visual consistency, and operational reliability. Jira roadmap data now links back to Jira, authentication failures sign you out cleanly, and the timeline view includes theme-aware styling with improved alignment and tooltips.

Key Updates

Gantt Timeline Enhancements

  • Theme-aware Fix Version bars (light/dark) for unreleased items.
  • Vertical timeline grid lines extend through the chart for easier scanning.
  • Hover tooltips show start/end dates for bars.
  • Milestone and UAT/Live marker alignment refined.

Jira Linking & Project Context

  • Fix Version and Epic names now link back to Jira.
  • Project key is included in the roadmap response for consistent navigation.

Authentication Reliability

  • Jira 401/403 responses now trigger a sign-out flow instead of a 500.
  • Session store uses timezone-aware UTC timestamps.

Release Notes UX

  • Release notes list and detail pages are accessible in-app.
  • Navigation between versions is included for quick reference.

How to Use

Timeline

  • Hover any bar to see precise start/end dates.
  • Select Fix Versions or Epics to open Jira links.

Sign-in / Sign-out

  • If Jira auth expires, you’ll be prompted to sign in again.
  • Use the header sign-out button to clear the session.

Notes

This release continues to build on v1 features (dashboards, rich-text panels, and filters). For full details, see v1 release notes.
