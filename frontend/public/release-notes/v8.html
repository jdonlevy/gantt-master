Delivery Tracker – Release Notes v8

Release Date Jun 5, 2026

Release Notes Index v7 v6 v5 v4 v3 v2 v1

Summary

v8 adds an initiative grouping layer above swimlanes, flexible bar colouring on the swimlane Gantt, and richer scoping for the Updates tab. The roadmap now loads almost instantly when filters haven't changed, and a number of UI rough edges have been polished.

Initiative grouping

  • Group swimlanes (or fix versions) under named initiatives, shown as a coloured left spine.
  • Initiatives can be collapsed/expanded, and the folded state is remembered across reloads.
  • Toggle the whole grouping layer on or off — existing dashboards render unchanged until you opt in.

Swimlane bar colours

  • New “Colour bars by” control: RAG status, Project, Swimlane, Initiative, or Manual.
  • Manual mode lets you define named colour categories and assign them by right-clicking a bar.
  • Add custom bars to a swimlane with their own name, dates, and colour.

Updates tab

  • The Updates summary is scoped by its own fix-version selection, independent of the roadmap.
  • New “Date from / Date to” filter sets a custom released window (defaults to the last two weeks).
  • Collapsing the filters, milestones, or custom-bars sections no longer flags unsaved changes.

Performance & polish

  • Roadmap loads instantly from a saved snapshot when filters are unchanged.
  • The active tab (Updates / Roadmap) is preserved across a page refresh.
  • Removed the sequential/concurrent dependency toggle — sequencing warnings are now always on.
  • Refined tab styling, the colour-mode hint, the Refresh button, and assorted spacing.
