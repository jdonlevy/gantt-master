Delivery Tracker – Release Notes v5

Release Date Apr 17, 2026

Release Notes Index v4 v3 v2 v1 Summary Key Updates How to Use Test Coverage

Summary

v5 tightens dashboard-scoped planning data and improves swimlane readability. Milestones and UAT/Live windows now save per dashboard, while schedule-based risk status and progress labels are visible in swimlane view. The Gantt header and legends move above the timeline with a compact horizontal layout.

Key Updates

UX Enhancements

  • Gantt header and legend are now positioned above the timeline.
  • Legend switches to a horizontal layout to reduce vertical whitespace.
  • Swimlane bars render behind progress pills for cleaner readability.

Status & Risk Visibility

  • Schedule-based status calculation now flags items as On track, At risk, or Overdue.
  • Progress percentages appear in swimlane labels.
  • Status colors are consistent between Standard and Swimlane views.

Dashboard-Scoped Planning Data

  • Milestones are now saved per dashboard (with optional project scoping retained).
  • UAT and Go Live windows are scoped per dashboard.
  • Swimlane view now renders UAT and Live markers for fix versions.

How to Use

  • Set UAT/Live windows on a dashboard and they will only appear on that dashboard.
  • Use Swimlane view for per-lane progress labels and schedule risk highlighting.

Test Coverage

  • Backend tests updated for dashboard-scoped milestones and overrides.
  • Frontend tests updated for dashboard-aware roadmap fetches.
If a milestone or UAT/Live window was created before v5, re-save it in the dashboard to associate it with the current dashboard scope.
