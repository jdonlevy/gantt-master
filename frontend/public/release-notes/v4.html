Delivery Tracker – Release Notes v4

Release Date Apr 3, 2026

Release Notes Index v3 v2 v1 Summary Key Updates How to Use Test Coverage

Summary

v4 introduces swimlane support in the Gantt, adds dashboard deletion, and tightens filter behavior so Fix Version selections apply everywhere. A new logging toggle keeps verbose progress JQL output off by default.

Key Updates

Swimlane Gantt

  • New Gantt view toggle (Standard / Swimlane).
  • Manual lane editor: name lanes and assign fix versions across projects.
  • Overlapping fix versions stack vertically inside a lane.
  • Fix Version filters now cascade into swimlanes and the lane selector list.
  • Lane hints show when selected fixes are hidden by the active filter.

Dashboard Management

  • Dashboards can now be deleted from the dashboard list (auth required).

Operational Control

  • Progress JQL logging is now behind a toggle: set LOG_PROGRESS_JQL=true to enable.

How to Use

  • Switch to Swimlane view in the Filters card, then assign fix versions to lanes.
  • Use Fix Version filters to limit what appears in swimlanes and lane pickers.
  • Delete dashboards from the list page when they are no longer needed.

Test Coverage

  • Frontend tests cover swimlane filtering behavior.
  • Backend tests cover dashboard deletion.
Swimlanes only show fix versions that pass the active Fix Versions filter. If a lane appears empty, verify the filter selection first.
